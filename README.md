# Agentic_AI_Receptionist
Project title:

AI-Powered Conversational Booking Assistant (Dialogflow)

Description (use something like this):

Built an AI-powered conversational agent using Dialogflow to automate customer booking workflows for a pet service business.
The system handles multi-step interactions, extracts structured booking data (service type, date, time, customer details), and simulates a real-world receptionist experience.

Key Features:
Multi-step booking flow (context-based conversation)
Entity extraction (date, time, service type, phone number)
Intent classification and routing
Conflict handling between FAQ and transactional intents
Scalable design for backend integration (API/webhook ready)
